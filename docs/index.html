O que é o CaraCore PDV?

É um programa de caixa que fica no computador da sua loja — não depende de nuvem para vender no balcão. Você registra vendas, controla o turno, acompanha estoque e exporta relatórios.

O que ele resolve: fila por internet instável, sistema pesado que trava o PC e a dor de trocar de PDV quando abre o segundo caixa. Comece em um PC; evolua para rede com o mesmo produto.

Rust vs Java — qual PDV escolher?

A Cara Core mantém duas linhas desktop. Esta vitrine é o PDV novo (Rust + Tauri, piloto Windows). O PDV Java é o canal maduro multi-plataforma.

Este PDV (Rust) — escolha se…

  • Opera no Windows e quer o piloto mais recente.
  • Prioriza programa leve e abertura rápida.
  • Quer testar grátis no PC da loja.

PDV Java — escolha se…

  • Precisa de Linux ou macOS.
  • Quer a linha v3.1.x já consolidada.
  • Já usa o PDV Java e quer continuidade.
