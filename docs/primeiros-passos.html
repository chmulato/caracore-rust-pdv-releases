Primeiros passos no Windows

Antes de abrir

  • Windows 10 ou 11 (64 bits).
  • Microsoft Edge WebView2 Runtime — na maioria dos PCs já instalado.
  • Pasta fixa para extrair o ZIP (ex.: C:\Programas\CaraCore-PDV\).

Instalação

  1. Baixe o ZIP em Download e extraia todo o conteúdo.
  2. Execute instalar-atalho.bat.
  3. Abra o atalho CaraCore PDV.
  4. Se o SmartScreen alertar, use “Mais informações” → “Executar assim mesmo”.

Primeira execução

  1. Conclua o assistente em /setup (banco SQLite local).
  2. Entre em /login com usuário, senha e código do caixa.
  3. A tela padrão após login é vendas em /vendas-v2.
  4. Configuração contínua: Preferências → Banco.

Onde ficam seus dados

ItemCaminho
Banco SQLite%APPDATA%\caracore-pdv\data\pdv.db
Configuração%APPDATA%\caracore-pdv\config.json
Logs%APPDATA%\caracore-pdv\logs\

Atalho: Win+R → digite %APPDATA%\caracore-pdv → Enter.

Operação offline

O PDV funciona sem internet. Vendas, caixa e estoque usam SQLite local. Funções fiscais remotas (SEFAZ) dependem de conectividade quando configuradas.
