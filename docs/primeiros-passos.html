Instalar e abrir o PDV

Passo a passo para a primeira execução no Windows — instaladores em português do Brasil.

Download

Início Produto Para sua loja Download Wiki

1. Baixar

Use apenas os releases oficiais no GitHub ou a página Download.

Confira o SHA256 conforme Transparência antes de executar qualquer arquivo.

  • Instalador (NSIS) — recomendado para lojista; assistente em português.
  • MSI pt-BR — recomendado para TI; arquivo *_pt-BR.msi.
  • ZIP portátil — extração manual; inclui LEIA-ME-INSTALACAO.txt em português.

2. Instalar

Opção A — Instalador NSIS (lojista)

  1. Execute o arquivo *-setup.exe.
  2. Siga o assistente em português até concluir.
  3. Abra CaraCore PDV pelo menu Iniciar ou atalho criado.

Opção B — MSI (TI / empresa)

  1. Instale o pacote *_pt-BR.msi (duplo clique ou msiexec /i "arquivo.msi").
  2. Para instalação silenciosa: msiexec /i "arquivo.msi" /qn.

Opção C — ZIP portátil

  1. Extraia todo o conteúdo em uma pasta fixa (ex.: C:\Programas\CaraCore-PDV\).
  2. Execute instalar-atalho.bat para criar atalho na Área de Trabalho.
  3. Abra CaraCore PDV pelo atalho.

3. Configurar

Na primeira abertura, o assistente orienta a preparação do banco de dados local. Depois, entre com usuário e senha fornecidos na implantação ou criados no setup.

4. Operar

Após o login, a tela de vendas fica disponível. Caixa, produtos e relatórios acessam-se pelo menu principal.

Onde ficam os arquivos

Dados e configuração ficam em %APPDATA%\caracore-pdv\. Para abrir essa pasta: tecla Windows + R, digite o caminho e confirme.
