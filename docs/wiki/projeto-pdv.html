Visão geral

O Cara Core PDV Desktop (Rust + Tauri 2) é um PDV desktop do mesmo CaraCore PDV: levar ao caixa as qualidades de Rust (segurança de memória, performance) e Tauri 2 (shell enxuto), com interface React e dados em SQLite local.

Release atual: v0.1.1 — Windows (NSIS/MSI pt-BR), Linux (deb/AppImage) e macOS (dmg). Inclui integração opcional com licença CaraCore Seed, checkout ágil, PIX, backup pós-turno e pós-venda por código. Faixa de versão distinta do PDV Desktop Java (canal v3.1.x).

Coexistência com PDV Desktop (Java)

A linha Java · JavaFX permanece a oferta madura para Windows, Linux e macOS (pdv.caracore.com.br, canal v3.1.2-free). Trilhas comerciais históricas, Wiki Fiscal e planos Free/Premium continuam na loja Java.

Escolha a iniciativa Rust + Tauri quando:

  • A loja participa do piloto multiplataforma (release v0.1.1).
  • A TI prefere MSI pt-BR ou quer avaliar Tauri/Rust.
  • O gestor aceita o escopo reduzido da fase piloto (ver Transparência).

Nenhuma linha substitui a outra. V3 (negócio) — posicionamento PME e PIX Split 2027 — é comum às duas stacks.
