Idioma dos instaladores

Os pacotes oficiais são gerados para o mercado brasileiro:

  • NSIS (*-setup.exe) — assistente em português do Brasil.
  • MSI (*_pt-BR.msi) — pacote WiX localizado para TI e deploy silencioso.
  • ZIP — aplicativo portátil; instruções em português no LEIA-ME-INSTALACAO.txt.

Requisitos

  • Windows 10 ou 11, 64 bits.
  • WebView2 Runtime — em geral já presente. Se necessário: download Microsoft.
  • SmartScreen: veja Transparência (comum sem certificado digital na fase piloto).
