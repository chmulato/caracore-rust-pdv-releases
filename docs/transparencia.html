Quem somos

O Cara Core PDV Desktop (Rust + Tauri) é desenvolvido e distribuído pela Cara Core Informática, CNPJ 23.969.028/0001-37.

Site institucional: caracore.com.br. Contato comercial e suporte: suporte@caracore.com.br.

Status do produto

Fase piloto (Windows) A iniciativa Rust + Tauri 2 (release v0.1.1) está em avaliação com lojas piloto em Windows, Linux e macOS. Não confunda com o PDV Desktop Java maduro em pdv.caracore.com.br.

Este site apresenta o PDV Desktop (Rust + Tauri) em fase piloto (release v0.1.1). Aqui você encontra benefícios, download multiplataforma e orientações para a loja.

Loja oficial

Esta loja em rust-pdv.caracore.com.br é a vitrine do PDV Desktop (Rust + Tauri 2). O download oficial está em GitHub Releases — a página Formatos só orienta NSIS, MSI e ZIP.

PDV Desktop Java maduro: pdv.caracore.com.br · Portfólio Cara Core: caracore.com.br.

Artefatos publicados

Cada release inclui instaladores localizados para o Brasil:

  • Instalador NSIS — assistente em português para o lojista.
  • MSI pt-BR — pacote para equipe de TI e instalação em rede.
  • ZIP portátil — uso sem instalador, com guia em português.
  • Lista de integridade — publicada junto a cada release para você conferir o arquivo baixado.

Fonte oficial de download

Baixe o pacote somente pelos releases publicados no GitHub:

github.com/chmulato/caracore-rust-pdv-releases/releases

  • Esta página de download apenas aponta para o mesmo endereço — não hospeda arquivos em outro domínio.
  • Desconfie de links enviados por terceiros, espelhos ou pastas compartilhadas sem confirmação com a Cara Core.
  • Se ainda não houver release publicado, o botão de download mostrará “Acompanhar releases” em vez de um arquivo inexistente.

Como conferir se o download é autêntico

Cada release oficial traz uma lista de verificação de integridade (algoritmo SHA-256). Depois de baixar o instalador ou ZIP:

  1. Abra a pasta do download no Windows.
  2. Gere a “impressão digital” do arquivo pelo PowerShell ou por ferramenta de hash indicada na própria release.
  3. Compare o resultado com o valor publicado na lista do release.

Se os valores não coincidirem, não execute o programa e avise suporte@caracore.com.br.

Assinatura digital e SmartScreen

Sem certificado de editor (por enquanto) O instalador piloto ainda não possui assinatura digital de código. O Windows SmartScreen pode exibir aviso na primeira execução. Isso é esperado nesta fase — não significa que o arquivo veio de outra fonte, desde que baixado do release oficial e conferido na lista de integridade.

Dados e privacidade

  • Produtos, vendas e caixa ficam no computador da loja, em pasta de dados do usuário Windows (perfil do operador).
  • O fluxo normal de operação local não envia o faturamento diário para servidores CaraCore.
  • Emissão fiscal eletrônica, quando configurada, comunica-se apenas com os serviços exigidos pela legislação (SEFAZ, conforme credenciais da loja).
  • Backups são responsabilidade da loja; o sistema oferece ferramenta local de cópia e restauração.

Escopo fiscal e funcional

O PDV inclui fila e rotinas fiscais em desenvolvimento. A homologação em ambiente real da sua loja depende de credenciais, certificado digital e orientação do contador. Recursos avançados (multi-loja, integrações específicas) seguem roadmap e contrato — não estão implícitos no pacote de teste.
