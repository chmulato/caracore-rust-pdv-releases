Visão rápida

Resumo dos pontos mais importantes sobre o Cara Core PDV Desktop:

  • Produto em fase piloto Windows (v0.1.2): Iniciativa em testes práticos e validação rápida em lojas piloto.
  • Download somente via GitHub oficial: Garantia absoluta de segurança ao obter o app da fonte correta.
  • Sem assinatura digital (SmartScreen): O Windows pode exibir um aviso de segurança no primeiro uso.
  • Dados 100% locais + backups automáticos: Soberania total sobre as vendas e backups em pendrives externos.
  • Edição gratuita limitada a 100 vendas: Licença de testes locais com teto máximo de cupons emitidos.
  • PIX manual no balcão: Sem verificação por webhook automático na versão gratuita.
  • Fiscal simulado (LOCAL_MOCK): Ambiente de treino fiscal por padrão no pacote sem custo.

Informações direcionadas por perfil

Explicamos os detalhes do nosso produto na linguagem certa para cada necessidade:

Para Lojistas

Foco no dia a dia da operação de vendas e caixa:

  • Frente de caixa offline: Seus dados não dependem de internet para registrar vendas locais.
  • PIX Manual (F9): Gere o QR Code na tela e confirme no comprovante do cliente antes de finalizar.
  • Limite de testes: Permite registrar até 100 vendas concluídas antes de exigir licença.

Para Equipes de TI

Instalação, integridade do sistema e segurança técnica:

  • Opções de pacotes: Escolha entre instalador NSIS, MSI corporativo ou portátil ZIP.
  • Impressão digital (SHA-256): Facilidade para comparar códigos e validar a integridade dos arquivos.
  • Aviso do Windows: Instruções claras para superar o aviso SmartScreen (sem assinatura de código por enquanto).

Para Contadores

Escopo fiscal, conformidade e custódia de dados:

  • Ambiente de treino (LOCAL_MOCK): A edição grátis não emite notas válidas e serve para simulações.
  • Conformidade LGPD: Criptografia de colunas de dados de clientes (PII) ativa diretamente no banco local.
  • Custódia exclusiva: A loja possui 100% da posse física e legal das movimentações e backups locais.

Como verificar a integridade do arquivo

Antes de executar o instalador ou ZIP, você pode verificar se o arquivo baixado é exatamente o original enviado pela nossa equipe. Pense no código SHA-256 como a impressão digital única do arquivo.

1. Baixar o arquivo Baixe o instalador desejado direto do repositório oficial no GitHub.
2. Gerar a impressão digital Use o PowerShell no Windows para gerar a chave única local do arquivo.
3. Comparar resultados Confronte a chave com o código publicado pela Cara Core na listagem do release.

Passo a passo no Windows (PowerShell)

  1. Abra a pasta onde você salvou o arquivo do download.
  2. Segure a tecla Shift, clique com o botão direito na pasta e selecione "Abrir janela do PowerShell aqui".
  3. Digite o seguinte comando (substituindo pelo nome do arquivo baixado) e pressione Enter: 
    Get-FileHash .\NomeDoArquivo.exe -Algorithm SHA256
  4. Confirme se o código gigante gerado na tela é igual à lista publicada na release correspondente do GitHub.
E se a impressão digital não coincidir? Se o código calculado for diferente do oficial, não execute o programa de forma alguma. Avise-nos imediatamente pelo e-mail suporte@caracore.com.br.

Avisos de segurança (SmartScreen)

Aviso do Windows SmartScreen Como esta versão está em fase piloto inicial, o instalador ainda não conta com um certificado digital corporativo pago (assinatura de código). Por esse motivo, ao executá-lo pela primeira vez, o Windows SmartScreen exibirá uma mensagem de alerta ("O Windows protegeu o seu computador").

Este comportamento é normal e esperado nesta etapa do piloto. Não representa riscos ao seu computador, desde que você tenha verificado a impressão digital do arquivo (SHA-256) na etapa anterior. Para continuar a instalação:

  1. Na janela do aviso do Windows, clique no link "Mais informações".
  2. Clique no botão "Executar assim mesmo" para iniciar o instalador oficial.

Privacidade e dados locais

Privacidade real e controle de dados. Nossas regras de tratamento de dados são transparentes e respeitam a LGPD:

✅ O que fazemos ❌ O que não fazemos
Dados 100% locais no seu PC: Suas vendas, estoque e movimentações financeiras ficam armazenados apenas na máquina da loja (no caminho %APPDATA%\caracore-pdv\). Sem envio de dados para servidores externos: A Cara Core não recebe logs de faturamento diário, tickets médios ou dados de clientes em nuvem centralizada.
Criptografia LGPD de clientes: Informações de clientes como Nome, CPF, Telefone e E-mail são criptografados localmente (cifra baseada em SHA-256) no SQLite para proteção contra cópia física direta. Sem telemetria oculta: Não rastreamos hábitos comerciais, preços praticados ou segredos de mercado do estabelecimento.
Logs de auditoria local contra fraudes: Registros permanentes no banco local identificam ações críticas do operador (exclusões e mudanças de parâmetros) com IP, hora e turno. Sem backup centralizado na nuvem: Como não coletamos dados, a custódia legal de backup físico para conformidade de faturamento é responsabilidade compartilhada exclusiva do lojista.
Comunicação fiscal segura direta: O app conversa direto com o webservice da SEFAZ para emitir cupons homologados, sem intermediários. Sem intermediação financeira: Transações de pagamento operam diretamente entre sua conta bancária e o cliente (na versão gratuita).

Backups automáticos físicos

O PDV executa uma rotina automática e silenciosa de backup no encerramento de cada turno do operador para prevenir perdas de dados:

1. Fechamento de caixa O operador aciona o encerramento do turno no aplicativo.
2. Busca de mídias O motor do PDV verifica pendrives e HDs externos conectados (letras D:\ a Z:\).
3. Cópia redundante O banco local de dados e logs são salvos de forma independente no dispositivo detectado.
Recomendação de segurança: Mantenha sempre um pendrive dedicado conectado ao computador do caixa para que as rotinas de backup físico ocorram com sucesso no final de cada dia.

Edição gratuita (Degustação)

O pacote básico permite que o lojista teste a agilidade e os fluxos do balcão sem pagar assinaturas.

⚠️ Limite Honestificado: Degustação de 100 Vendas A Edição Gratuita permite realizar até 100 vendas concluídas no banco local. A partir da 101ª venda, o sistema bloqueia novas movimentações até a ativação de uma licença Premium ou Seed. O app mostra avisos de progresso sobre o teto de cupons restantes.

O que está incluído na Edição Gratuita:

  • Relatórios financeiros resumidos no app para fechamento de caixa.
  • PIX com QR Code estático na tela (confirmação manual).
  • Ambiente fiscal simulado (LOCAL_MOCK) para treinos operacionais rápidos.

Para obter um plano comercial com suporte dedicado e remover o limite, envie um e-mail para suporte@caracore.com.br.

PIX no checkout

⚠️ PIX Manual (Sem Webhook Bancário) Na Edição Gratuita, o faturamento via PIX é totalmente manual. O app gera o QR Code na tela para o cliente escanear, mas não se comunica com a conta da sua loja para receber avisos automáticos de depósito. A conferência do recebimento é dever do operador de caixa.

Fluxo operacional do PIX manual (F9) no balcão:

1. Apresentar o QR Code O operador escolhe PIX no checkout e o sistema exibe o QR Code estático ou gera o Pix Copia e Cola.
2. Pagamento do cliente O cliente lê o QR Code pelo aplicativo do banco dele e realiza a transferência.
3. Checagem do caixa O cliente exibe o comprovante. O operador valida o recebimento no banco da loja e confirma manualmente a baixa no PDV.

Observação: A integração bancária direta por webhook (baixa automática na hora) é exclusiva do plano Premium.

Escopo fiscal e faturamento

⚠️ Ambiente de Treino Fiscal (LOCAL_MOCK) Na Edição Gratuita, a fila fiscal e a impressão rodam no modo LOCAL_MOCK (fiscal simulado). Nenhum dado é enviado para a SEFAZ. Isto permite realizar testes e treinar operadores sem gerar obrigações tributárias reais ou desperdiçar papel fiscal.
