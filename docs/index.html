O que é

O CaraCore PDV reúne venda no balcão, controle de caixa, cadastro de produtos e relatórios do dia. Os dados ficam na máquina da loja. Não é necessário manter o caixa dependente de nuvem para registrar vendas.

Para quem

  • Pequenas lojas de bairro, mercados e varejo local.
  • Quem prefere operar com software licenciado e dados sob controle próprio.
  • Quem quer testar o fluxo de caixa antes de ampliar licenças ou integrações.
