Como funciona por baixo dos panos

Rust — o motor

Linguagem compilada que gera binário nativo. Menos surpresas de memória e performance previsível no núcleo do caixa — vendas, estoque, caixa e fiscal.

Tauri 2 — a janela

Shell desktop leve que usa o WebView2 do Windows. Interface moderna sem carregar um navegador inteiro — startup rápido e instaladores profissionais.

SQLite — seus dados

Banco local na máquina da loja. Vendas e movimentos ficam sob seu controle, com backup automático ao fechar turno.

React — a tela

Interface operacional pensada para o balcão: busca rápida, atalhos de teclado e telas responsivas para gestão.

Distribuição profissional

  • NSIS — instalador com assistente em português para o lojista.
  • MSI WiX pt-BR — pacote para TI e deploy silencioso.
  • Download verificado — cada release traz lista para conferir o instalador.
  • Releases rastreáveis — versões publicadas de forma aberta no GitHub.
