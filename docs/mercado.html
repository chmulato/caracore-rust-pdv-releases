Seus dados

Produtos, vendas e movimentos de caixa ficam no computador da loja. Você decide quando fazer backup. Não é necessário enviar o faturamento diário para um servidor externo para manter o caixa aberto.

Licenciamento

  • Edição de teste — para conhecer o fluxo de venda e caixa em operação real, com limites definidos na proposta comercial.
  • Edição completa — amplia recursos e suporte conforme contrato com a Cara Core.

Valores e condições são informados no contato comercial. O download desta página aponta para releases oficiais no GitHub — consulte Download para saber se já há pacote publicado.

Internet e fiscal

Vendas e fechamento de caixa seguem sem conexão. Emissão de documentos fiscais eletrônicos depende de link com a SEFAZ quando configurada; o sistema mantém fila local e retoma o envio depois.

Outras edições

A Cara Core também oferece o PDV na linha Java, com distribuição multiplataforma, em pdv.caracore.com.br. Esta loja trata da linha v3 para Windows descrita neste repositório.
