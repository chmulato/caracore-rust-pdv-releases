Valor da stack desktop

Para o gestor e a TI, o PDV Desktop Rust + Tauri traduz engenharia de ponta em benefícios comerciais concretos:

  • Controle total — software licenciado instalado na máquina da loja; dados locais em SQLite.
  • Menor pegada — Tauri 2 entrega shell desktop enxuto; ideal para PCs de balcão modestos.
  • Confiança operacional — Rust traz previsibilidade de memória e performance no núcleo do caixa.
  • Distribuição profissional — MSI/NSIS em português, verificação SHA256, releases rastreáveis.

Seus dados

Produtos, vendas e movimentos de caixa ficam no computador da loja. Você decide quando fazer backup. Não é necessário enviar o faturamento diário para um servidor externo para manter o caixa aberto.

Licenciamento

  • Edição de teste — para conhecer o fluxo de venda e caixa em operação real, com limites definidos na proposta comercial.
  • Edição completa — amplia recursos e suporte conforme contrato com a Cara Core.

Valores e condições são informados no contato comercial. O download desta página aponta para releases oficiais no GitHub — consulte Download para saber se já há pacote publicado.

Internet e fiscal

Vendas e fechamento de caixa seguem sem conexão. Emissão de documentos fiscais eletrônicos depende de link com a SEFAZ quando configurada; o sistema mantém fila local e retoma o envio depois.

Outras edições

A Cara Core mantém duas linhas do mesmo produto CaraCore PDV: esta loja trata da iniciativa Rust + Tauri 2 (release v0.1.0, piloto Windows). O PDV Desktop Java (canal v3.1.x, multi-plataforma) está em pdv.caracore.com.br — faixas de versão independentes, sem conflito.
