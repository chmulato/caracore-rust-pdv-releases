Instalar e abrir o PDV

1. Baixar

Use apenas os releases oficiais no GitHub ou a página Download.

Confira a integridade do arquivo conforme Transparência antes de executar qualquer instalador.

  • Instalador NSIS — recomendado para lojista; assistente em português.
  • MSI pt-BR — recomendado para equipe de TI.
  • ZIP portátil — extração manual com guia em português no pacote.

2. Instalar

Opção A — Instalador NSIS (lojista)

  1. Execute o instalador baixado (assistente em português).
  2. Siga as telas até concluir.
  3. Abra CaraCore PDV pelo menu Iniciar ou atalho criado.

Opção B — MSI (TI / empresa)

  1. Instale o pacote MSI pt-BR (duplo clique ou ferramenta de deploy da sua TI).
  2. Para instalação silenciosa em rede, use o procedimento padrão de MSI da sua equipe de TI.

Opção C — ZIP portátil

  1. Extraia todo o conteúdo em uma pasta fixa no computador da loja.
  2. Use o atalho ou script de instalação incluído no pacote, se disponível.
  3. Abra CaraCore PDV pelo atalho.

3. Configurar

Na primeira abertura, o assistente orienta a preparação do banco de dados local. Depois, entre com usuário e senha fornecidos na implantação ou criados no setup.

4. Operar

Após o login, a tela de vendas fica disponível. Caixa, produtos e relatórios acessam-se pelo menu principal.

Onde ficam os arquivos

Dados e configuração ficam no perfil do Windows do usuário que opera o PDV. Se precisar localizar a pasta, a equipe de suporte Cara Core orienta conforme sua instalação.
