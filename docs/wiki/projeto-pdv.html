Visão geral

O Cara Core PDV v3 é a nova linha do ponto de venda CaraCore, construída em Rust com shell desktop Tauri 2 e interface React. Os dados ficam em SQLite na máquina da loja.

Release piloto atual: v0.1.0 (Windows x64, instaladores em português do Brasil).

Stack técnica

CamadaTecnologia
Core / domínioRust (workspace Cargo)
DesktopTauri 2 (WebView2 no Windows)
UIReact + TypeScript
PersistênciaSQLite local
DistribuiçãoWiX MSI pt-BR, NSIS, ZIP portátil, SHA256SUMS, RELEASE_MANIFEST.json

Coexistência com PDV Desktop (Java)

A linha Java · JavaFX permanece a oferta madura para Windows, Linux e macOS (pdv.caracore.com.br). Trilhas comerciais históricas, Wiki Fiscal e planos Free/Premium continuam na loja Java.

Use o PDV v3 Rust quando:

  • A loja participa do piloto Windows assistido.
  • A TI prefere MSI pt-BR ou quer avaliar Tauri/Rust.
  • O gestor aceita o escopo reduzido da fase piloto (ver Transparência).
