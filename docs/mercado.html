Seus dados

Produtos, vendas e movimentos de caixa ficam no computador da loja. A build de 02/06/2026 pode gerar backup automático do banco ao fechar o turno — o gestor não precisa lembrar de copiar o arquivo manualmente todo dia. Restauração e cópias extras continuam disponíveis nas telas de suporte.

Planos e licenciamento

Modelo de ativo licenciado — você adquire o software para a loja, com opção de mensalidade de suporte. Valores finais conforme proposta comercial.

Todos os planos incluem: instaladores pt-BR (NSIS, MSI), dados locais na loja, operação offline para vendas e caixa, e verificação SHA256 nos releases. Solicite proposta comercial →

Internet e fiscal

Vendas e fechamento de caixa seguem sem conexão. Emissão de documentos fiscais eletrônicos depende de link com a SEFAZ quando configurada; o sistema mantém fila local visível e retoma o envio depois, com mensagens claras para o operador.

Outras edições

A Cara Core mantém duas linhas do mesmo produto CaraCore PDV. Esta loja trata da linha Rust + Tauri (piloto Windows). O PDV Desktop Java (canal maduro, multi-plataforma) está em pdv.caracore.com.br.
