Processo

Como usar esta loja

Feito para lojista e gestor de loja — instalação em Primeiros passos e Download.

  • Antes de operar: na primeira execução use o assistente /setup (banco); depois entre em /login com usuário e senha. Configuração contínua em Preferências → Banco.
  • No balcão: após login, a tela padrão é vendas (/vendas-v2). Consulte Diferenciais e Aparência.
  • Para decidir investimento: veja Para sua loja e o manual completo do produto.
  • Abrir localmente: abra a página inicial desta loja no navegador — todos os links funcionam offline.
