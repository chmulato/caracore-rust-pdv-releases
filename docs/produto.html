Tecnologia desktop

O Cara Core PDV Desktop (Rust + Tauri) combina um núcleo em Rust com shell Tauri 2 no Windows — arquitetura pensada para lojas que querem software instalado, leve e auditável.

  • Binário nativo enxuto — menos overhead que stacks pesadas de desktop; startup rápido no balcão.
  • Segurança de memória (Rust) — reduz classes inteiras de falhas que afetam sistemas de caixa 24/7.
  • Instaladores pt-BR — MSI WiX e NSIS localizados para TI e lojistas brasileiros.
  • Release reprodutível — manifesto SHA256 e pipeline dedicado oficina → loja.

Operação

  • Instalado no Windows da loja; funciona offline para vendas e caixa.
  • Cada operador entra com usuário e senha.
  • Temas de tela claro e escuro, conforme preferência.
  • Atalhos de teclado (F1 ajuda, F2 busca, F9 finalizar) para agilizar o balcão.
  • Cancelamentos com motivo registrado — trilha para auditoria e gestão.

Observações

A homologação fiscal em ambiente real depende das credenciais e do acompanhamento do contador. Recursos avançados de rede multi-loja entram conforme evolução do produto e contrato.
