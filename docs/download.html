Conteúdo do ZIP

  • Programa principal (caracore_pdv_desktop.exe)
  • Script para criar atalho na Área de Trabalho
  • Arquivo de verificação (SHA256)
  • Instruções de instalação em texto

Requisitos

  • Windows 10 ou 11, 64 bits.
  • Componente WebView2 — em geral já presente no Windows atualizado. Se necessário: download Microsoft.
  • Se o Windows SmartScreen alertar sobre o arquivo, use “Mais informações” e “Executar assim mesmo” (comum enquanto o instalador não possui certificado digital).

Instalação resumida

  1. Extraia o ZIP em uma pasta fixa (ex.: C:\Programas\CaraCore-PDV\).
  2. Execute o script de atalho.
  3. Abra o CaraCore PDV e siga o assistente inicial.
